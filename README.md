# Metriq

Metriq is a proof-of-work hiring platform MVP.

## Repo structure

- `apps/web`: Next.js (App Router) frontend
- `packages/ui`: shared design system and reusable components
- `packages/types`: shared TypeScript domain types
- `packages/validators`: shared Zod validation schemas
- `packages/config`: shared TS config (and other shared config as needed)

## Getting started

1. Install dependencies:

```bash
pnpm install
```

2. Start web app:

```bash
pnpm dev
```
